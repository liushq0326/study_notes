# 一、设计模式

## 1.1 概念
  >在面向对象软件设计过程中针对特定问题的简洁而优雅的解决方案。
  >通俗一点说，设计模式是在某种场合下对某个问题的一种解决方案。

  >在软件设计中，模式是一些经过了大量实际项目验证的优秀解决方案。熟悉这些模式的程序员，对某些模式的理解也许形成了条件反射。当合适的场景出现时，他们可以很快地找到某种模式作为解决方案。

  >所有设计模式的实现都遵循一条原则，即“找出程序中变化的地方，并将变化封装起来”。
